## 🔌 Основні компоненти коду (Приклади)

### 🧩 NT-Xent Loss (Функція втрат)
```python
def nt_xent_loss(z1, z2, temperature=0.2):
    # Об'єднуємо два views та рахуємо косинусну схожість
    z = torch.cat([z1, z2], dim=0)
    sim = torch.mm(z, z.T) / temperature
    # Заглушуємо діагональ матриці (self-similarity)
    mask = torch.eye(z.size(0), dtype=torch.bool).to(z.device)
    sim.masked_fill_(mask, float('-inf'))
    return F.cross_entropy(sim, target)
```

## 🧪 Проблеми і рішення

| Проблема | Рішення |
| :--- | :--- |
| **CUDA Out of Memory (OOM)** | Зменшити `batch_size` (наприклад, з 256 до 128) або увімкнути змішану точність (mixed precision). |
| **Розрив з'єднання з Colab** | Зберігати checkpoints моделі кожні 5-10 епох на Google Drive, щоб відновити навчання з останньої точки. |
| **Помилка розміру батча при NT-Xent** | Обов'язково встановити `drop_last=True` у DataLoader, щоб гарантувати парну кількість зображень для матриці схожостей. |
